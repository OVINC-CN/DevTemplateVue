## Vue3 Template

Vue3 Template for OVINC Services

## Quick Start

### 1. Install `cruft`

```bash
$ pip install cruft
```

### 2. Initial Project

```bash
$ cruft create https://github.com/OVINC-CN/DevTemplateVue.git --checkout main
```

### 3. Prepare Env

```bash
$ export HOST=
$ export BACKEND_URL=
$ export SITE_URL=
$ export OVINC_URL=
$ export OVINC_WEB_URL=
```

## Install Dependecies

```bash
$ npm install pnpm -g
$ pnpm i
```

## Run Service

```bash
$ pnpm run dev
```
